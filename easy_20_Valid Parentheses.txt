20. Valid-Parentheses

** 문제내용

'(', '), '{', '}', '[' 및 '] 문자만 포함된 문자열이 주어지면 입력 문자열이 유효한지 확인하십시오.
입력 문자열은 다음과 같은 경우에 유효합니다.

- 열린 괄호는 같은 유형의 괄호로 닫아야 합니다.
- 열린 브래킷은 올바른 순서로 닫아야 합니다.
- 모든 폐쇄 브래킷에는 동일한 유형의 개방 브래킷이 있습니다.


예시 1)
Input: s = "()"
Output: true


예시 2)
Input: s = "()[]{}"
Output: true


예시 3)
Input: s = "(]"
Output: false


제약 )

1 <= s.length <= 104
s consists of parentheses only '()[]{}'.
