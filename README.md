## ✔ **커밋 규칙**
* 파일명 : 난이도_문제번호_문제이름.js
  * ex) medium_2_Add Two Numbers.js

* 커밋 메세지 : 난이도 / 문제번호 / 문제이름
  * ex) medium / 2 / Add Two Numbers
 

## 👀 **코드리뷰**
* 자신의 브랜치에 commit 및 원격에 push 
* 풀 리퀘스트(PR) 생성
* PR명:  자유

스터디 시간에 PR code review 후 merge해서 아카이브할 예정
